# dotfiles

Neovim (Kickstart.nvim) and kanata keyboard remapper configuration.

## What's Included

- **nvim/** - Neovim config (Kickstart.nvim + lazy.nvim)
- **kanata/** - Keyboard remapper config

## Quick Setup

### Neovim
```bash
ln -s ~/.dotfiles/nvim ~/.config/nvim
```

### Kanata
```bash
ln -s ~/.dotfiles/kanata/.config/kanata ~/.config/kanata
```
